## Install Wyvern
<CodeGroup>

```shell pip
pip install wyvern-ai
```

```shell poetry
poetry add wyvern-ai
```

This will also install the [wyvern CLI](/references#wyvern-cli)

</CodeGroup>

## Create A New Wyvern ML Application
1. Navigate to the directory where you want to create the Wyvern application.
2. Once Wyvern is installed, run this command and follow the instructions to set up a new repository for your Wyvern application
```
wyvern init
```

Here are the prompts that need your initial inputs:
- `Your project name`: enter the name of your project

Now that the `wyvern init` CLI has set up your initial repository, you should see the following file structure in your repository:
```
├── pipelines
│   ├── __init__.py
│   ├── main.py
│   ├── product_ranking
│   │   ├── __init__.py
│   │   ├── models.py
│   │   ├── ranking_pipline.py
│   │   ├── realtime_features.py
│   │   ├── schemas.py
├── tests
│   ├── __init__.py
├── pyproject.toml 
├── ci
│   ├── deploy.py
├── .env
└── .gitignore
```

## Run Your Wyvern ML Application

### Pre-requisite
1. Redis service

Wyvern uses redis as it's index and feature store. By default, Wyvern connects to the localhost:6379

Your can run this command to install and spin up local redis
```
wyvern redis
```

### Run the wyvern service
```
wyvern run
```

Wyvern spins up a fastapi service with the default ranking API `/api/v1/ranking`. Refer to http://0.0.0.0:5001/redoc#operation/MyRankingPipeline_api_v1_ranking_post for the ranking schema.
